export
    특정파일(모듈)안에 있는 내용(함수,변수,컴포넌트,....)을 다른 파일에서
    사용하고자 한다면 export를 해야지만 사용할 수 있다

    이때 export하는 방법은 두가지 방법이 있다
        1. export하고자하는 함수나 변수 혹은 컴포넌트 앞에다가 export 키워드 붙이기

            export function Hello(){

            }

        2.파일에 직접 명시하기
            export default 이름;
                파일 당 단 하나만 기본으로 내보내기 할 수 있다
            export {내보낼 요소 / 이름[as 별칠],이름[as 별칭],이름};
                여러개의 요소를 중괄호로 묶어서 내보내기 할 수 있다

import
    외부 파일에 있는 내용(함수, 컴포넌트,변수,...)을 현재 파일에서 사용하고자 할 때 
    import하여 사용할 수 있다. 관례적으로 맨 윗쪽에 추가하여 가독성을 높인다
        import {이름,이름,이름,...} from 파일경로/라이브러리;

    이때 이름은 만약 export한 쪽이 별칭을 설정했다면 별칭대로 적어준다
    별칭을 설정하지 않았다면 export된 이름 그대로 적어준다

    또한 해당 파일이나 라리브러리에서 default로 export한 녀석은 중괄호를 생략하고 import한다
        import 이름 from 파일경로/라이브러리

    이때 이름은 export 된 그대로 사용해도 되고, 우리가 별칭을 붙여서 사용해도 된다

페이지 추가하기
    경로를 설정하고 싶다
    SPA(Single-Page-Application)은 html문서가 1개이다
    버튼이나 글자를 클릭했을때 보여지는 페이지는 사실 다른 페이지가 아니라
    다른 js파일 속 콤포넌트이다.
    서로다른 컴포넌트 들이 마치 독립적인 페이지 처럼 보이는 것이며
    이를 도와주는 라이브러리가 react-router-dom 이다

react-router-dom 라이브러리
    BrowserRouter
        HTML5의 History API를 활용하여 페이지를 새로고침 하지 않고도
        주소를 변경할 수 있도록 도와주는 컴포넌트

    Routes
        url이 변경되었을 때 자식 컴포넌트인 Route를 보며 어떤 컴포넌트와
        매칭해야 하는지 판단하는 역할을 한다
        자식으로 Route 컴포넌트가 있어야 한다
    Route
        url과 컴포넌트를 매칭하고,url이 변경되면 해당 컴포넌트를
        랜더링 하는 역할 
        path에는 url상의 원하는 주소를 문자열로 전달,
        element에는 해당 주소 입력시 렌더링 되어야 하는 컴포넌트(페이지)를
        중괄호 안쪽에 작성
        ex) <Route path = '/add  element ={<컴포넌트 />} />


node.js로 자바스크립트 실행하기
    실행시키고자 하는 자바스크립트 파일이 들어있는 폴더로 이동
        node 파일이름
    이때,확장자는 제외하고 입력하면 결과가 terminal에 출력된다


비구조화 할당

    구조화된 데이터(객체,배열)를 구조를 해체하여 변수에 할당하는 문법

    객체
        객체 속 키값과 같은 이름의 변수에다가 각각을 풀어서 할당할 수 있으며
        이때, 변수들은 중괄호로 묶어야 한다
    배열
        배열 속 요소들이 인덱스 번호 순서대로 변수에 대입이 되며
        변수들의 이름 제약은 없으나 순서가 존재하고, 각 변수들은 모두
        대괄호를 통해 묶어야 한다

props 
    상위 컴포넌트의 값을 하위 컴포넌트에 전달할 변수를 의미하는 말
    해당 함수형 컴포넌트의 매개변수가 부모 컴포넌트로부터 ket와 value를 받아온다

state
    컴포넌트 자기 자신이 갖고 있는 값, 화면상 변해야 하는 부분
    react 라이브러리에 있는 useState() 함수를 사용해야하며
    useState() 함수사용 결과는 초기값, state 변수에 값을 대입할 수 있는 함수가
    들어있는 배열이다

    보통은 state 변수이름과 함수를 짝궁지어 비슷하게 짓는다
    ex) num setNum
        cnt setCnt
        name setName

        setState() 함수는 인자에다가 state변수에 대입하고자 하는 값을 전달할 수도 있고,
        함수를 전달할 수도 있다
        setState()를 통해 state변수 값이 변하면 re-rendering을 한다
